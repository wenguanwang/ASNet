===========================================================================

1. Download our model from https://drive.google.com/file/d/16RSGrRe_0cj_owSLbmpbSYgnqRxMZii_/view?usp=sharing

2. Run "main.py".

3. The computed saliency maps are stored in the "saliency_predictions" folder.
